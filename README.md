# cait.yutube# VibeStream — Платформа Видеостриминга

Инновационный и адаптивный веб-интерфейс видеохостинга, построенный на базе **React** и **Vite**.

## Особенности проекта
- Уникальный дизайн, свободный от копирования чужих товарных знаков.
- Встроенная динамическая темная и светлая тема.
- Полная адаптивность: от больших 4К мониторов до мобильных устройств.
- Автоматический поиск и фильтрация по категориям без перезагрузки страниц.
- Архитектура без бэкенда: использует статический `mockData.json`.

## Локальный запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com<YOUR_GITHUB_USERNAME>/vibestream.git
   cd vibestream
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

## Публикация и Автоматический деплой

В репозиторий уже добавлен GitHub Actions workflow. Для публикации вам нужно:
1. Заменить `<YOUR_GITHUB_USERNAME>` в файлах `package.json` и `vite.config.js` на ваш реальный никнейм GitHub.
2. Сделать коммит и отправить код в ветку `main`:
   ```bash
   git add .
   git commit -m "feat: Инициализация проекта и CI/CD"
   git push origin main
   ```
3. GitHub Actions автоматически соберет проект и создаст ветку `gh-pages`.
4. В настройках вашего репозитория (**Settings -> Pages**) убедитесь, что источником сборки выбрана ветка `gh-pages`.
